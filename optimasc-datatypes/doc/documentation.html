Introduction

This datatype library permits to manage properties of data types, as well as to create data types., where a datatype is defined as a set of distinct values, characterized by properties of those values, and by operations on those values [1].

The datatypes are characterized by different properties which are similar across schemas and languages.

Basic datatype properties

Bounded

A datatype is bounded when it has a lower and upper bound.

Ordered

A datatype is ordered, when a value of that datatype can be ordered in comparison to other values of that datatype. For example, for integers, the value 1 is smaller than the value 10.
