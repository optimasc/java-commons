Package overview

This package contains classes that can be used to retrieve information from archive formats as well as extract information from archive files. It could also be used to represent disk images.

How to use

The ArchiveEntry interface is used to represent an entry in an archive file or disk file, while the ArchiveFileReader is an abstract implementation of an archive reader that must be implemented for each archive format that shall be supported.

The ArchiveEntry interface is similar to org.apache.commons.compress.archivers.ArchiveEntry interface of Apache common compress.
